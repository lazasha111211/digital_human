# 文生视频功能描述



- 上载视频文件，保存到指定的 uploads\vedio目录

- 点击生成，调用本地，提取视频中音频中的文字，显示进度条，返回的内容显示在 vedio_text_output 输入框中

- 对返回的内容进行描述，

- 点击生成，调用comfyui 工作流，显示进度条，对返回的内容的按照输入的描述要求进行加工后，获得返回的内容还是显示在 vedio_text_output 输入框中


- 选择引用的音频上载，保存到指定的 uploads\audio目录

- 点击生成，利用vedio_text_output 中的文字 和上载音频 ，调用 comfyui 工作流，显示进度条，最后合成配音文件

- 解析返回的内容，下载配音文件到 outputs \audio目录，播放配音文件


- 选择人物正脸图片上载，保存到指定的 uploads\image目录

- 点击生成，利用上载图片和上面合成的配音文件，调用comfyui 工作流生成视频，显示进度条

- 解析返回的内容，下载视频文件到 outputs\gifs目录，播放视频
